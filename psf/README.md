The format of the example PSF coefficient file (`psf_ccfbrd210048.bin.txt`) is:
```
hw
ndeg_spat
ndeg_local
ngauss
recenter
cos
sin
ax
ay
sigma_inc
sigma_mscale
fitrad
x_orig
y_orig
vec_coeffs (till EOF)
```

More information about this is in DIAPL package's [documentation](http://users.camk.edu.pl/pych/DIAPL/diapl2.pdf)
